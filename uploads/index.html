วิธีเรียนให้เข้าใจจริง

  1. อ่านทฤษฎีก่อน — แต่ละหน้าเริ่มด้วย "หลักคิด" สั้น ๆ อย่ารีบข้ามไปดูโค้ด ถ้ายังไม่เห็นภาพในหัว
  2. กด Play อนิเมชัน — ดู input ที่ตัวอย่างให้มาก่อน 1 รอบ แล้วลองเปลี่ยน input เอง (เช่น เปลี่ยนขนาด array, เปลี่ยนกระดาน N-queens) ถ้าเดาไม่ออกว่าจะเกิดอะไร = ยังไม่เข้าใจ
  3. อ่านโค้ด ทีละบรรทัด — แต่ละหน้ามีโค้ด C++ ครบ เปิดไฟล์ในโฟลเดอร์ lab/, labSortLab2/, labWeak2/, tranning/ ของจริงคู่ไปด้วย แล้ว trace ตามมือ
  4. ทำแบบฝึก — ทุกหน้ามี exercises พร้อม <details> ปิดเฉลยไว้ พยายามทำเองก่อน 10 นาที ค่อยเปิดดู
  5. เขียน-รัน ในเทอร์มินัล: 
    g++ -std=c++17 -Wall myfile.cpp -o run && ./run

โน้ต C++ ที่ใช้ในทุกตัวอย่าง

โค้ดในเว็บนี้เขียน style เดียวกับไฟล์ในโฟลเดอร์ (มี using namespace std;, ใช้ global variable, บางที่ใช้ 1-indexed เพื่อให้ตรงกับ pseudocode ในเลกเชอร์)

เครื่องมือ C++ ที่ใช้บ่อยใช้ทำอะไร
vector<int> v;array ที่ขยายขนาดได้ — ใช้แทน array แบบเก่าเสมอ
v.push_back(x);เติม element ท้าย vector
v.size()คืนจำนวน element (เป็น size_t — ระวังเปรียบเทียบกับ int)
swap(a, b)สลับค่า a, b — เร็วกว่าเขียน temp เอง
sort(v.begin(), v.end())เรียงน้อย→มาก, complexity O(n log n)
cin >> x;รับ input
cout << x << endl;print + ขึ้นบรรทัดใหม่
เทคนิคอ่านโค้ดเร็ว: เริ่มที่ main() ก่อนเสมอ ดูว่ามันเรียกฟังก์ชันอะไร แล้วค่อยตามไปอ่านฟังก์ชันนั้น อย่าอ่านจากบนลงล่างเรียง ๆ จะหลง
